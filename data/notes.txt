EvalIQ — AI-Powered Automated Grading System
Final Year Project • UMT 2024–Present • Team: Junaid Iqbal, Husnain Shahbaz

An end-to-end system for automated subjective answer grading. Uses TrOCR to extract handwritten student responses, Sentence-BERT for semantic similarity scoring, FLAN-T5 with LoRA for personalised feedback, and a BERT classifier for Bloom's Taxonomy categorisation. Built on a Django + React + PostgreSQL + Celery stack with planned GCP deployment.
